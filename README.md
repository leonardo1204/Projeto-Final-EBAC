Dashboard de Performance Comercial: Insights Olist
Projeto Final - EBAC (Escola Britânica de Artes Criativas e Tecnologia)

🎯 Objetivo do Projeto

Este dashboard foi desenvolvido como projeto de conclusão do curso na EBAC. O objetivo principal é transformar dados brutos do ecossistema de e-commerce da Olist (2016-2018) em inteligência de negócio, permitindo a análise multidimensional de vendas, logística e comportamento de categorias.

📊 Estrutura do Dashboard

O relatório foi construído seguindo uma hierarquia de informações para facilitar a tomada de decisão:

KPIs Principais: Visualização imediata de Faturamento Total, Ticket Médio e Prazo Médio de Entrega.

Série Temporal: Gráfico de linha detalhando a evolução das vendas ao longo dos meses.

Análise de Mercado (Treemap): Mapa de árvore que exibe a representatividade e concentração de mercado por categoria de produto.

Eficiência Logística: Ranking de prazos médios por estado, identificando gargalos operacionais.

Ranking de Categorias: Identificação dos pilares de receita através de gráficos de barras verticais.

🛠️ Tecnologias e Funcionalidades

Looker Studio: Ferramenta utilizada para a visualização e design do dashboard.

Campos Calculados: Implementação de lógica via fórmulas (como a extração de YEAR e métricas de Ticket Médio) para enriquecer a análise.

Interatividade: Inclusão de filtros de Ano (2016-2018) e Categorias, permitindo que o usuário explore os dados de forma dinâmica.

Data Storytelling: Legendas explicativas integradas para guiar o usuário na interpretação técnica dos gráficos.
